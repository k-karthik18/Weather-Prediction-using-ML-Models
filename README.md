## 📊 Project Overview
This project focuses on comparing traditional machine learning algorithms against an ensemble approach for day-wise rainfall prediction. By combining multiple predictive models, the system successfully mitigates the weaknesses of individual algorithms, resulting in a highly accurate and robust prediction engine.

### Key Features
* **Extensive Data Processing:** Cleaned and processed 10,000+ historical weather records for training and testing.
* **Multi-Model Implementation:** Built and evaluated four distinct ML architectures:
  * Decision Tree
  * Random Forest
  * Support Vector Machine (SVM)
  * Gradient Boosting
* **Ensemble Optimization:** Applied ensemble modeling techniques, which improved overall prediction accuracy by **12%** compared to the individual base models.
* **Data Visualization:** Integrated 3+ dynamic graphs to visualize:
  * Temperature trends over time
  * Rainfall distribution patterns
  * Model accuracy and performance comparisons
